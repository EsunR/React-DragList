# React拖拽组件

用户向组件传入一个数组作为数据源，通过拖拽可以调整数据在数组中的位置。

## 数据源格式

```js
[
  { data: "数据1", id: 1 },
  { data: "数据2", id: 2 },
  { data: "数据3", id: 3 }
]
```

## 组件参数

| 参数    | 说明             | 类型   | 默认值 |
| ------- | ---------------- | ------ | ------ |
| data    | 数据源           | Array  | 必填项 |
| style   | 自定义的样式     | Object | {}     |
| spacing | 拖拽项之间的空隙 | Number | 0      |
